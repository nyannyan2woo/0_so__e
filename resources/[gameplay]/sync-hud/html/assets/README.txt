║                     LOGO INSTRUCTIONS                         ║
╚═══════════════════════════════════════════════════════════════╝

Place your server logo in this folder named "logo.png"

RECOMMENDED:
- Size: 64x64 or 128x128 pixels
- Format: PNG with transparency
- Keep it simple - it displays small

The logo will automatically appear in the top-right status bar
with the animation you set in config.lua

Available animations in config.lua:
- "none"    - Static logo
- "pulse"   - Subtle scale/fade pulse
- "glow"    - Glowing effect
- "rotate"  - Slow rotation
- "bounce"  - Gentle bounce

Example:
Config.LogoAnimation = "pulse"
